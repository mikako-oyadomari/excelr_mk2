Excelr mk2
======

エクセルからデータを抜き取り、htmlを生成します。  
このExcelr mk2では、一つのhtmlにエクセルから読み取ったデータを出力することが可能です。


Feature
-------

- エクセルデータを読み込み
- テンプレートへ流し込み


Modules
-------

- **[Exceljs](https://github.com/guyonroche/exceljs)** : エクセルファイルを取り込む
- **[Nunjucks](https://mozilla.github.io/nunjucks/)** : テンプレートエンジン
